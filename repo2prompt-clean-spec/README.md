# Repo2Prompts Specs

This folder stores product and acceptance specs only. It is not the runnable
Next.js app.

Use the application repo root one level up:

```bash
cd ..
npm run bootstrap
```

If you already opened this folder in a terminal, these commands forward to the
repo root and print a reminder:

```bash
npm run bootstrap
npm run dev
npm run test
npm run build
```
